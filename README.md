# cricket-fever
This repository is used to build cricket game application. 
This application will consist of following features : 
- Player Mangement
- Team Management
- Player Statistics
- Match Statistics
- Overall Statistics

Tech Stack used for the developement is : 
Go Langauage , Postgres , Prometheus , Grafana, Jaeger .
